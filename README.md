## fisher

### Install
```sh
curl https://git.io/fisher --create-dirs -sLo ~/.config/fish/functions/fisher.fish
```

### Config
```sh
$ ln -s ~/configurations/fish/config.fish ~/.config/fish
$ ln -s ~/configurations/fish/fishfile ~/.config/fish
```
