## homebrew

### Install
https://brew.sh/

### Packages
```sh
$ brew install fish git mise fzf ripgrep fd eza bat zoxide ghq tig starship
$ brew install --cask font-jetbrains-mono-nerd-font
```

## fisher
```sh
curl -sL https://raw.githubusercontent.com/jorgebucaran/fisher/main/functions/fisher.fish | source && fisher install jorgebucaran/fisher
```

## Config
```sh
$ ln -s ~/configurations/fish/config.fish ~/.config/fish/
$ ln -s ~/configurations/fish/fish_plugins ~/.config/fish/
$ ln -s ~/configurations/ghostty ~/.config/
$ ln -s ~/configurations/vim/.vimrc ~/
```
